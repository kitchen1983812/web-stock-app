# モバイルブラウザでの動作確認手順

スマートフォンからこのアプリ（WebStock）を利用・テストするための手順書です。

## 前提条件
- PCとスマートフォンが**同じWi-Fi（ローカルネットワーク）**に接続されていること。

## 手順

### 1. サーバーの起動 (PC側)
ターミナルで以下のコマンドを実行します。

```bash
npm run dev:host
```

または、SSLエラー（「この接続はプライバシーが保護されていません」）を回避したい場合は、以下を実行します（HTTPモード）。

```bash
npm run dev:http
```

### 2. IPアドレスの確認
コマンド実行後、ターミナルに以下のような表示が出ます。

```
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.1.15:5173/  <-- これを使います
```

`Network` の右側に表示されているURL（例：`http://192.168.1.15:5173/`）をメモしてください。

### 3. スマートフォンでのアクセス
スマートフォンのブラウザ（Chrome, Safariなど）を開き、手順2で確認したURLを入力してアクセスします。

#### SSL警告が出る場合 (`npm run dev:host` の場合)
`https://...` でアクセスすると、自己署名証明書のため警告画面が出ます。
- **Chrome/Android**: 「詳細設定」→「...にアクセスする（安全ではありません）」をタップ。
- **Safari/iOS**: 「詳細を表示」→「Webサイトを閲覧」をタップ。

### 4. 動作確認
以下の点が改善されているか確認してください。
- 画面の左右の余白が適切か（広すぎないか）。
- 文字の大きさが読みやすいか。
- 「ページを追加」ボタンやリストの表示が崩れていないか。
