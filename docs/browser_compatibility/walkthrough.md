# モバイルブラウザ対応 ウォークスルー

web-stock-appをスマートフォンで利用可能にするための修正を行いました。

## 変更内容

### 1. レスポンシブデザインの適用
**ファイル**: `src/index.css`
- PC表示では `2rem` だったパディングを、モバイル（画面幅640px未満）では `1rem` に縮小し、画面を有効活用できるようにしました。

### 2. モバイルデバッグ用スクリプトの追加
**ファイル**: `package.json`, `vite.config.http.js`
- ローカルネットワーク内のスマホから簡単にアクセスできるよう、以下のnpmスクリプトを追加しました。
    - `npm run dev:host`: ホストアクセスを許可して起動（SSLあり）
    - `npm run dev:http`: ホストアクセスを許可して起動（SSLなし、httpのみ）

## 検証方法

1.  PCでコマンドプロンプトを開き、プロジェクトフォルダに移動します。
2.  `npm run dev:host` または `npm run dev:http` を実行します。
3.  表示される `Network: http://...` のURLを確認します。
4.  同じWi-Fiに接続したスマートフォンで、そのURLにアクセスします。
5.  画面のレイアウトが崩れていないことを確認します。

## 注意事項
- `npm run dev:http` (HTTPモード) では、PWAのインストールや一部の共有機能（Share Target API）が動作しない場合があります。本番環境（HTTPS）であれば問題ありません。
